Tinetti Balance and Gait Evaluation (POMA)

Balance Subscale (0–16)

1. Sitting Balance
0 = Leans or slides; 1 = Steady, safe
2. Arises
0 = Unable without help; 1 = Uses arms; 2 = Without arms
3. Attempts to Arise
0 = Unable without help; 1 = More than one attempt; 2 = One attempt
4. Immediate Standing Balance
0 = Unsteady; 1 = Steady with support; 2 = Steady without support
5. Standing Balance
0 = Unsteady; 1 = Steady with support; 2 = Steady without support
6. Eyes Closed
0 = Unsteady; 1 = Steady
7. Nudge on Sternum
0 = Begins to fall; 1 = Steady with support; 2 = Steady without support
8. 360° Turn
0 = Unsteady/discontinuous steps; 1 = Continuous; 2 = Steady
9. Sitting Down
0 = Unsafe; 1 = Uses arms or not smooth; 2 = Safe and smooth

Gait Subscale (0–12)

10. Initiation of Gait
0 = Hesitant; 1 = No hesitation
11. Step Length & Height
0 = Uneven; 1 = Normal
12. Step Symmetry
0 = Asymmetrical; 1 = Symmetrical
13. Step Continuity
0 = Discontinuous; 1 = Continuous
14. Path
0 = Deviates; 1 = Straight
15. Trunk
0 = Sways or uses aid; 1 = No sway or device
16. Walking Stance
0 = Heels apart; 1 = Heels nearly touching
Total Score: 0 / 28
Fall Risk: Low
(25–28 = Low, 19–24 = Moderate, <19 = High)
